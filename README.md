# Nestio
# 🏠 Nestio – Vacation Rental Platform

Nestio is a full-stack web application inspired by modern vacation rental platforms like Airbnb.
It allows users to discover unique places to stay, book properties, and host their own listings.

The platform provides a seamless experience for travelers to find comfortable accommodations while enabling hosts to list and manage their properties.

---

## 🚀 Features

### 👤 User Authentication

* Secure signup and login
* Password encryption
* Session-based authentication

### 🏡 Property Listings

* Add new property listings
* Upload property images
* Edit or delete listings
* View property details

### 🔍 Search & Explore

* Browse available stays
* View location and property information
* Explore different accommodations

### ⭐ Reviews & Ratings

* Users can leave reviews
* Rating system for listings

### 📍 Map Integration

* View property locations on interactive maps

### 📅 Booking System

* Select check-in and check-out dates
* Book available properties

## 🚀 Getting Started

Follow these simple steps to set up and run the project on your local machine.

### 1️⃣ Clone the Repository

Download the project from GitHub:

```bash
git clone https://github.com/Rony117-png/Nestio.git
```

### 2️⃣ Navigate to the Project Directory

Move into the project folder:

```bash
cd Nestio
```

### 3️⃣ Install Dependencies

Install all required packages and dependencies:

```bash
npm install
```

This command will automatically create the `node_modules` folder and install everything needed to run the project.

### 4️⃣ Start the Application

Run the project using:

```bash
npm start
```

### 🎉 You're All Set!

The application should now be running successfully on your local machine.

### 📌 Notes

- Ensure that **Node.js** and **npm** are installed on your system.
- Run `npm install` whenever you clone the project on a new device.
- The `node_modules` folder is not included in the repository and will be generated automatically during installation.
- If you encounter any issues, delete `node_modules` and run `npm install` again.
